# Repository Baseline

## Required Root Files

- `CLAUDE.md` — Claude Code project instructions
- `AGENTS.md` — Multi-agent repository guidelines
- `CODEX.md` — Codex/generic agent guidelines
- `CHANGELOG.md` — Keep a Changelog 1.1.0
- `CONTRIBUTING.md` — Contribution guidelines
- `CODE_OF_CONDUCT.md` — Contributor Covenant
- `SECURITY.md` — Security policy
- `SUPPORT.md` — Support channels
- `LICENSE` — GPLv3
- `README.md` — English readme
- `.editorconfig` — Editor configuration
- `.gitattributes` — Git attributes
- `codecov.yml` — Coverage configuration
- `sonar-project.properties` — SonarCloud configuration

## Required Directories

- `.github/` — Workflows, issue templates, PR template, CODEOWNERS, dependabot
- `.claude/skills/` — Repository-local skills
- `docs/en/` — English documentation (authoritative)
- `docs/ru/` — Russian documentation (translation)
- `docs/plans/` — Sprint plans, ROADMAP.md, BACKLOG.md
- `scripts/` — Build, test, quality, release scripts
- `deploy/podman/` — Container configurations

## Skills Alignment

Keep `CLAUDE.md`, `AGENTS.md`, `CODEX.md`, and `.claude/skills/` aligned when rules change.
